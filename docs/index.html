egg-sendmail

Sendmail plugin for Egg.js.

Install

npm i egg-sendmail --save

Usage

// {app_root}/config/plugin.js
exports.sendmail = {
  enable: true,
  package: 'egg-sendmail',
};

Configuration

// {app_root}/config/config.default.js
exports.sendmail = {
};

see config/config.default.js for more detail.
