# egg-sendmail

[Nodemailer](https://nodemailer.com) Egg.js插件。

## 安装插件

```bash
npm i @axolo/egg-sendmail --save
```

## 开启插件

```js
// config/plugin.js
exports.sendmail = {
  enable: true,
  package: '@axolo/egg-sendmail',
};
```

## 详细配置

请到 [config/config.default.js](config/config.default.js) 查看详细配置项说明。
